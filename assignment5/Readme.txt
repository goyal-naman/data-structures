=====================
Introduction
=====================

The project is application of undirected graphs data structure using C++.

=====================
Compiling and Running
=====================

First of all locate to root directory of the project

*To compile the code:
make

*To run the code:
./make


*To clean:
make clean

=====================
Executing the code
=====================

Follow appropriate on-screen instructions while executing the code. Choose the desired option and input the corresponding arguments.

Countries are listed as
Country Name > Neighbour 1 > Neighour 2 > ...

For border of country larger than X functionality, border is taken as sum of borders of for all neighbouring countries.
